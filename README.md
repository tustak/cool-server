# Cool (QL) server

**Right now it is working with a MongoDB local server**

## Installation

Clone
```
git clone git@github.com:tustak/cool-server.git
```

Enter folder
```
cd cool-server
```

Install
```
npm install
```

## Run

For running GraphQL server
```javascript
npm run build
npm start
```
